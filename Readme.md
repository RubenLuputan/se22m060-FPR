### Adding a new Order

- 'AddOrder' [`products`] 'Category' [`categories`]
- 'AddOrder' [`products`] 'Category' [`categories`]

> `products` and `categories` are strings separated by a blank space
>  
> Example: AddOrder `Flour Eggs Milk` Category `Baking Ingredients`
